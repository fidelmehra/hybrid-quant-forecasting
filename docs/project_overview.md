# Project Overview

## Objective

Benchmark hybrid financial time-series forecasting models against classical baselines across multiple asset classes and market regimes.

## Scope

- Asset classes: Bitcoin, equities, macro-linked instruments
- Models: ARIMA, GARCH, Prophet, XGBoost, RF, LightGBM, LSTM, GRU, Temporal CNN, hybrid combinations
- Evaluation: RMSE, MAE, directional accuracy, regime-wise performance, Sharpe/drawdown

## Key Contribution

Systematic comparison of econometric, ML, and DL forecasting approaches with regime-aware evaluation — no synthetic data.
